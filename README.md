# Claves_Master v1.0
**Proyecto Integrador - Lógica de Programación**

## 1. Introducción y Objetivo
Claves_Master es una solución integral para la gestión de seguridad de usuarios. El objetivo principal es proporcionar una herramienta robusta que permita registrar usuarios en una base de datos centralizada (SQL Server), autenticarlos de forma segura y generar contraseñas aleatorias de alta complejidad evaluadas mediante un algoritmo de seguridad en tiempo real.

## 2. Funcionalidades Principales
* **Módulo de Autenticación:** Validación de credenciales contra SQL Server.
* **Registro de Usuarios:** Formulario dinámico para la creación de nuevas cuentas con validación de duplicados.
* **Generador Dinámico:** Creación de claves aleatorias basadas en configuraciones almacenadas en la DB.
* **Semáforo de Seguridad:** Evaluación lógica de la fortaleza de la clave (Rojo/Naranja/Verde).
* **Gestión de Portapapeles:** Función para copiar la clave generada al sistema operativo.

## 3. Tecnologías Utilizadas
* **Lenguaje:** Python 3.12+
* **Interfaz Gráfica:** Tkinter
* **Base de Datos:** SQL Server 2022 (vía pyodbc)
* **Arquitectura:** Modular (Separación de lógica de inicio y generador)
